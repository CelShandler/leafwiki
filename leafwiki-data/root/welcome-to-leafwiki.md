# Bem-vindo ao LeafWiki!

LeafWiki – Uma wiki rápida para quem pensa em pastas, não em feeds.
Um único binário em Go. Markdown armazenado em disco. Sem serviços de banco de dados externos.

O LeafWiki é uma wiki leve e auto-hospedada para *runbooks*, documentação interna e notas técnicas — criada para escrita ágil e estrutura clara. Ele armazena seu conteúdo como arquivos Markdown simples em disco e oferece navegação, busca e edição rápidas, sem a necessidade de executar serviços adicionais.


---

## Recursos

- Páginas baseadas em **Markdown** armazenadas em disco (sem necessidade de banco de dados)
- **Navegação hierárquica** com seções e páginas
- **Busca de texto completo** (full-text search) com tecnologia SQLite FTS5
- **Gerenciamento de arquivos** (upload, renomeação e exclusão de anexos)
- **Histórico de revisões** com snapshots e restauração
- **Importação** a partir de arquivos ZIP contendo Markdown
- **Personalização de identidade** (nome do site, logotipo, favicon)
- **Multiusuário** com controle de acesso baseado em funções (administrador / editor / visualizador)
- **Modo de acesso público** para navegação anônima somente leitura

## Primeiros Passos

1. Crie sua primeira página usando o botão **+** na barra lateral
2. Escreva em **Markdown** — suporte a títulos, listas, blocos de código e links
3. Use **seções** para agrupar páginas relacionadas em uma hierarquia semelhante a pastas
4. Faça upload de arquivos arrastando-os para o editor

Para mais informações, visite o [repositório do LeafWiki no GitHub](https://github.com/perber/leafwiki).
